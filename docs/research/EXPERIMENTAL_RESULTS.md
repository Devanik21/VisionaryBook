# Experimental Results

## Executive Summary
This document logs the outcomes of our benchmark tests evaluating Xylia's core capabilities.

## Accuracy Metrics (Preliminary)
- **Botanical Identification**: 94% Top-1 Accuracy on controlled dataset.
- **Landmark Recognition**: 91% Accuracy, highly dependent on viewing angle.
- **Educational Diagrams**: 88% accuracy in correctly identifying sub-components.

## Performance Metrics
- **Average API Latency**: 3.2 seconds.
- **UI Rendering Latency**: < 50ms.
- **Audio Synthesis**: ~1.5 seconds per paragraph.

## Observations
- Confidence scores provided by the model correlate strongly with actual accuracy.
- Lighting and resolution significantly impact extraction quality in non-ideal conditions.
