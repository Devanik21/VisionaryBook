# Future Work & Roadmap

## Overview
Strategic objectives and research directions for the next iterations of Xylia.

## Short-Term Goals (Next 6 Months)
- Implement caching mechanisms for redundant image queries to save API costs.
- Develop a dedicated mobile application frontend.
- Enhance local TinyDB storage with advanced search and indexing capabilities.

## Long-Term Research Objectives
- **Local On-Device Models**: Investigate running quantized vision models locally to reduce API dependency and improve privacy.
- **Video Stream Analysis**: Extend the pipeline to support real-time video frame processing.
- **Personalized Pedagogy**: Adapt the flashcard generation algorithm to learn from user performance over time, tailoring difficulty automatically.
